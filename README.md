# Atlanta-OSS

Atlanta-OSS is a web application aimed at controlling and stabilizing the data contribution of small businesses.

## Project Description

Atlanta-OSS provides a platform for small businesses to contribute and manage their data effectively. The goal is to help these businesses maintain consistent and accurate data, which can be beneficial for their growth and operations.

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm (v6 or later)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Loxyez/Atlanta-OSS.git
```

2. Navigate to the project directory:
```bash
cd atlanta-oss
```

3. Install the dependencies:
```bash
npm install
```

## Running the Application
To start the development server, run:

```bash
npm run start:local
```
To develop in local environment when developer want to develop API geteway from local core lambda

```bash
npm run start:dev
```
To allow developer that want to implement UX/UI only

### This will start the React application on http://localhost:3000.
